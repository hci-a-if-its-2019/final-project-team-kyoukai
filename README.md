Subject: [javatpoint.com](javatpoint.com)

First Impression:
  1. The homepage is bulky
  2. Overwhelming informations
  3. No internal search bar

## Transcript 1
```text
Participant's description:
1. Participant is a 20 years old woman of Indonesian origin.
2. Participant is an undergraduate student, currently studying in ITS.
3. Preferred using mobile over desktop for casual browsing.
4. Participant frequently access internet for 12 hours on daily basis.
5. Participant never access a programming tutorial site.

Author: Ramdan
Note:
A: Me
B: Participant

1. A: Aku mau kamu mencoba satu web, kemudian kamu lakukan suatu tugas.
2. B: Oke.
3. A: Bagaimana kalau begini, menurutmu tampilan seperti ini bagus seperti kamu mau mencari tutorial begitu.
4. B: Ini tutorial apa saja?
5. A: Semuanya, jadi banyak banget.
6. B: Bisa jelaskan mungkin, ini ada sub tema begitu, misalnya ini, ini buat apa

(Participant refers to the category titles such as "Popular Tutorials", "Latest Tutorials", and "B.Tech / MCA" thus I explain first three categories.)

7. A: Anggaplah kamu mencari sesuatu di web, seperti mencari bahasa tertentu, sebagai pengguna biasa kamu cepat mencarinya?
8. B: Sebenarnya kalau sudah mengetahui apa yang mau digunakan itu cepat, cuma ini ada beberapa misalnya aku mau mencari cara A dan ternyata ada cara B yang lebih baik penggunaanya, hanya karena tidak ada penjelasan (kategori) jadi aku tidak tahu ini apa, dan itu membingungkan.
9. A: Kemudian, aku minta kamu coba mencari, disini kan ada banyak bahasa, aku beri nama bahasanya, C++ (Called 'C Tambah Tambah' (C++) for Participant's convinience), coba kamu cari.

(Participant tried to search 'C++' in the search bar, since the search bar utilize Google Custom Search, no results point to javatpoint.com)

10. B: Di hasil ini, banyak.
11. A: Ya, di aplikasi ini, bukan ke web lain.

(The focus of the interview is the website, not the search results)

12. A: Aku mau carinya yang masih di website javatpoint.com.
13. B: Berarti di javatpoint.com.

(Participant searching manually on the website)

14. A: Ya harus kamu cari manual berarti.
15. B: Ya, aku harus cari manual, karena disini tidak bisa control f kan?
16. A: Ya (tidak bisa)

(The participant access the website using a mobile phone  (but still accessing the desktop version so it is still valid) and is unaware of the 'Search in page' function of the browser.)

17. B: Susah ya jadinya (You don't say, milady)
18. B: C++ itu simbolnya apa (Referring to C++ logo in javatpoint.com), oh ini ya (Participant complete the task after a significant time searching).
19. A: Itu masih gampang kan, karena ada di atas. Misal aku cari yang bawah?
20. A: Hanya itu yang masih kuperlukan. Sekarang aku mau tau impresi pertama kamu bagaimana sebagai pengguna yang belum pernah mencari seperti ini dan saran kamu mengenai web ini.
21. B: Kategori ini. Misalnya yang B.Tech / MCA ini apa, intinya bagi aku yang dari diluar jurusan programmer aku tidak mengerti ini apa, itu gunanya buat apa, dan ini itu apa.
22. B: Sebenarnya kalau melihat yang seperti ini kan gak enak dilihat mata kan, orang biasanya lebih suka bentuk pola gambar ini.
23. B: Atau gak campurkan aplikasi sebagai kelompok. Misalnya Excel, Word, Powerpoint jadi Microsoft Office, gak usah disebar begitu. Kan orang sudah tau Microsoft Office itu apa saja isinya.

(Participant refers to how the tutorials are clustered according to spesific/scientific terms, and not to layman's terms. Thus, she suggest to cluster the tutorials according to layman's terms and categories.)
```
